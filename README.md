# gan_basic
gan_basic is a python implementation of several generative adversarial network.

## Requirements
- Tensorflow 1.0+

## Features
- 1dim_gaussian is GANs for one dimentional gaussian.
- mnist is small DCGANs for mnist.
- face is DCGANs for face images.
